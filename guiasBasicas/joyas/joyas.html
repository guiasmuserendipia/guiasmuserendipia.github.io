JOYAS

Obtención →

Pueden caer en cualquier mapa cuando se mata cualquier mob (bicho), u obtenerlo mediante eventos: Skeleton invation, Red Dragon Invation, White Wizard invation, Blood Castle, Devil Square.


Utilidad y otras características según joya

Se utilizan para subir ítems desde cero hasta +6, para diferentes combinaciones y para curarle la vida al fenrir.

Se utilizan para subir ítems desde +7 hasta +9, y para diferentes combinaciones.

Se utilizan para añadir la opción de adicional (ADD) a cualquier ítem (ya sean armas, alas o partes del set), para distintas combinaciones y/o para venderlas al NPC por zen.

Se utilizan para diferentes combinaciones como por ejemplo crear invitaciones para Devil Square, Blood Castle, crear y evolucionar Alas, subir ítems, etc.

Se utilizan para diferentes combinaciones, como por ejemplo para crear Fenrir, caballo para el DL (DarkHorse), y Alas Evo 3. También poniendolas en packs de 10, 20 y 30 pueden cambiarse por silver coins en el NPC de Lorencia.

Se utiliza para crear una Jewel of Ascent.

Esta semi-joya únicamente se la puede conseguir en Atlans o en Box of Luck.

Jewel of Ascent

Puede crearse con la Gemstone o se puede comprar en la tienda con Goldcoins.


Sirve para agregar entre 1 y 51 puntos extra para distribuir a tu personaje. Solo pueden utilizarla los personajes con 300 resets. El máximo de puntos por personajes es de 30 mil puntos extras.


Como fabricarla:

Para crearla tenes que ir a la Chaos Machine y poner una Gemstone y una Jewel of Chaos.

La probabilidad fija es de un 70% de éxito.

Esta joya SOLO se puede conseguir matando los Bosses (White Wizard, Red Dragon, Skeleton King, Kundun).

Se utiliza para subir ítems +14 y +15
